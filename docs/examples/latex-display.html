Display of a LaTeX expression selected from an array:

Since backslashes are used in JavaScript to escape characters, the simplest way to avoid problems and save typing is to double them in all LaTeX expressions. Then they are replaced by an HTML entity that renders as a single backslash in the browser. Note also that MathJax needs to be told that a part of the page has changed and needs to be typeset.
